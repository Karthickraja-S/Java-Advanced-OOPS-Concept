Dynamic Questions : 
1. Does we have limited obstacles like snakes, ladders OR need to handle extra obstacles? -> Future Obstacles need to added
2. Do we need to have a exact count of snakes & ladders / extra can be added ? -> Dynamic
3. Does dice is limited / N no of dice can be used -> N number
4. Do we have limited no of players / dynamic -> Dynamic
5. Board size is constant / dynamic -> Dynamic

Entities based on the above question :

Board , Cell , Obstacle ( Abstract ) , ObstacleFactory , 
ObstacleType , Snake , Ladder , Dice , Player , Game.
